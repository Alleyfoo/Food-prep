## CLI proof

Show the first protocol:

```text
foodprep protocol roasted_broccoli_component lingonberry_vinegar
```

Record the result only after performing the physical test:

```text
foodprep record-tasting roasted_broccoli_component lingonberry_vinegar \
  --verdict promising \
  --preparation "roasted floret, vinegar added warm" \
  --ratio "1 floret : 3 drops" \
  --temperature warm \
  --observations "..." \
  --failure-mode "..." \
  --successful-correction "..." \
  --supporting-ingredients "sea_salt" \
  --safety-confirmed
```

The accepted verdicts are: `works`, `promising`, `works_only_in_this_form`,
`needs_adjustment`, `dominated`, `texturally_wrong`, `clashes`, and
`inconclusive`.
