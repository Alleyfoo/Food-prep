## Known issues / limitations
- Missing-role logic is the weakest empirically-labeled layer (per the report);
  it is hand-curated, not corpus-derived. This is intentional for the MVP.
- Availability is an approximate Finnish/Lidl estimate, not live stock.
- Prompt parsing is regex keyword-based; handles the brief's canonical prompts
  but is not a full NLU. The "can" technique keyword uses a negative lookahead
  to avoid collision with the English modal "can".
- `confit` and `fermented tomato condiment` branches are deliberately excluded
  from the MVP (report recommends a later safety-aware phase).
